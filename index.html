React ES6 業界開發實戰應用課程

React.js 是一個由 Facebook 團隊所開發維護，開源的 Javascript 網頁框架。以它的模組化、生命周期設計、優秀的效能等，深受前端工程師開發者的喜愛。

但入門困難是 React 公認的缺點，尤其官方與市面上多數教學都建構在 nodejs、webpack、ES6 之上，使得新手無所適從。

本 React + ES6 課程會從最簡單的 React 語法入門教學，手把手的方式實作範例，一步一步穩紮穩打，循序漸進的讓你學會 React，進入現代前端工程師開發的領域。

Vue.js 前端框架 - 業界開發實戰應用課程

Vue.js 是一套用於構建用戶界面的漸進式框架，不但易於上手，還便於與其他第三方庫或即有專案整合。因其優秀的效能與平緩的學習曲線，在近幾年已攻佔前端開發的市場，與 React 並列目前人氣最高的兩大前端框架。

本課程教學使用 Vue.js 2.x 版本，將透過完整的教學範例，讓你一步步深入了解 Vue 的運做原理。相較於 React，Vue 的最大特點是可以直接用於網頁，不需要編譯也很適合後端開發人員使用
